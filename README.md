ТРАНСПОРТНЫЙ АНАЛИЗАТОР (Консоль v3.0 - Стратегическое Планирование)

Описание Проекта

Транспортный Анализатор (v3.0) — это настольное приложение, разработанное на Python с использованием библиотеки PyQt5, предназначенное для стратегического анализа транспортных потоков и автоматического генерирования рекомендаций по нивелированию негативных последствий от градостроительного развития или текущих проблем УДС (улично-дорожной сети).

Система использует улучшенную ИИ-модель v3.0 для расчета комплексного Индекса Серьезности, классификации проблемного участка по ТИРам (1-4) и выбора наиболее оптимального мероприятия из библиотеки действий, основываясь на критерии "Эффективность / Стоимость".

Основные Возможности

Загрузка и Симуляция Данных: Автоматическая загрузка данных из файлов .geojson в текущей директории и симуляция дополнительных факторов (RoadClass, WeatherImpact, PredictiveLoad).

Расчет Нагрузки и Приоритетов: Комплексный анализ текущей и прогнозируемой нагрузки с учетом структурных особенностей (ширина, наличие перекрестка, управление светофором).

Тирирование Проблем (ТИР 1-4): Классификация участков по четырем уровням приоритета (от Критического до Планового Контроля).

Стратегическое Планирование (ИИ v3.0):

Генерирование развернутой HTML-рекомендации для выбранного участка.

Выбор оптимального мероприятия (Minor, Medium, Major) для максимизации снижения нагрузки.

Отчетность: Поддержка (опционально) генерации PDF- и Excel-отчетов для всего набора данных.

Требования к Запуску

Для запуска приложения требуется Python 3 и следующие библиотеки: PyQt5, reportlab, openpyxl

Приложение ищет файлы с расширением .geojson рекурсивно во всех подпапках. Для корректной работы каждый объект (feature) в GeoJSON должен содержать в блоке properties следующие поля:

Поместите файлы .geojson в ту же директорию, где находится скрипт traffic_analyzer.py, или в любую его подпапку.


Запустите скрипт:

python traffic_analyzer.py


Работа с Интерфейсом:

В таблице (СЕГМЕНТ 01) отображаются исходные данные, включая симулированные факторы (RoadClass, WeatherImpact) и прогнозную нагрузку (PredLoad).

Используйте выпадающий список "ВЫБОР УЧАСТКА", чтобы выбрать интересующий сегмент.

Нажмите кнопку "СТРАТЕГИЧЕСКИЙ АНАЛИЗ ИИ", чтобы получить детальный анализ (ТИР, Индекс Серьезности) и оптимальную рекомендацию по мероприятию.

Результаты анализа отобразятся в СЕГМЕНТЕ 03.
